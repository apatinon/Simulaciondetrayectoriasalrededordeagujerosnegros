# Simulacion de trayectorias alrededor de agujeros negros
Se usa python y RK de cuarto orden para calcular trayectorias de haces de luz cerca de agujeros negros
no cargué las imagenes y videos generados porque el archivo quedaba muy pesado.

Basicamente definimos haces de luz que dejan estelas detrás de ellos y calculamos las trayectorias que siguen alrededor del agujero negro, esta simulación muestra una de dos opciones para cada fotón: Se destruye al caer en el agujero (pérdida de información física) o su trayectoria se ve curvada dependiendo de qué tan fuerte es la atracción.
Este es un ejemplo básico de cálculo de trayectorias con pasos, hay que tener en cuenta que el agujero negro es muy simple, no rota, la deformación espacial alrededor de él es constante, no varía en el tiempo.

Lo ideal sería poder usar constantes reales y elevar la simulación a cosas más complejas.
